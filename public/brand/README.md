# Gorilla Smash Club Brand Assets

## Logo Files

This directory contains the official branding assets for Gorilla Smash Club.

### Current Files:
- `logo.svg` - Full horizontal logo with text (600×200px)
- `logo-mark.svg` - Icon only, no text (200×200px)
- `gorilla-smash-full.png` - Full featured logo with tagline (recommended for main display)

## How to Add Your Logo

1. **Save your logo image** in this directory as:
   - `gorilla-smash-full.png` (for PNG format)
   - or `gorilla-smash-full.webp` (for WebP format - smaller file size)

2. **Recommended specifications**:
   - Format: PNG (with transparency) or WebP
   - Size: 800×400px or similar aspect ratio
   - Resolution: 144 DPI or higher for retina displays
   - Background: Transparent or matches your theme

3. The system will automatically use this logo throughout the website.

## Usage

The logo appears in:
- Header (navigation bar)
- Home page hero section
- Footer
- Authentication pages
- Custom appearance settings (when user uploads custom logo)

## Customization

Users can upload their own custom logos via the Appearance Settings page at `/settings/appearance`.
